Description
This project is a simple hospital management system that manages patients based on priority. Patients with higher urgency are treated first using a priority
queue approach, ensuring efficient and organized patient handling.

Features
1) Add patient details
2) Assign priority based on urgency
3) Serve patients in priority order
4) Display patient queue
5) Efficient and structured patient management

Tech Used
C- language 

Concepts Used
1)Priority Queue
2)Data Structures
3)Basic Algorithms

How to Run
1) Compile the program
2) Run the executable file

Project Goal

The goal of this project is to understand and implement the concept of priority queues in a real-world application. It demonstrates how hospitals can manage
patients more effectively by treating critical cases first, which helps improve efficiency, reduce waiting time, and ensure better organization in patient handling systems.
